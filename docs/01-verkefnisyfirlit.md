# Kotrusamband Íslands — Verkefnisyfirlit

> Síðast uppfært: 3. mars 2026

## Um verkefnið

Kotrusamband Íslands (KSÍ) er að byggja upp stafrænt vistkerfi fyrir íslenska kotrumenningu á léninu **kotra.is**. Verkefnið nær yfir vefsíðu, spilarakerfi, mótastjórnun, fræðslu og samfélag.

## Lykilákvarðanir

| Þáttur | Ákvörðun |
|---|---|
| **Lén** | kotra.is (bíður rétthafaréttinda) |
| **Tungumál** | Tvítyngd (IS/EN) |
| **Gagnagrunnur** | Já — dynamískt efni |
| **Ytri þjónustur** | BG Galaxy, BG Studio Heroes, BG NJ |
| **Póstlisti** | Listmonk (sjálfshýst) |
| **Forum/samfélag** | Discord server |
| **Workspace** | Google Workspace Business Starter á @kotra.is |
| **Útlit** | Fjólublátt + gull, hlýlegt en prófessíónelt |
| **WBIF aðild** | Staðfest — Ísland er skráð á wbif.net |

## Ákveðið — Tæknistaður ✅

| Þáttur | Val |
|---|---|
| **Framework** | Astro 5.x (static-first, island architecture) |
| **Backend** | Supabase (BaaS — PostgreSQL, Auth, Storage) |
| **CSS** | Tailwind CSS 4.x |
| **Interactive** | Svelte 5 (islands þar sem þarf) |
| **Hosting** | Cloudflare Pages (eða Vercel) |
| **i18n** | astro-i18n |
| **Type safety** | Supabase auto-generated types (ekkert ORM) |

## Óákveðið enn

- Lógóhönnun
- Fjöldi Google Workspace notenda
- Listmonk VPS hýsing (hvar)
