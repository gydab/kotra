## Síðulýsingar

### Forsíða (/)
- Hero section með kynningu
- Fréttir / tilkynningar
- Næstu viðburðir/mót
- Fremstu spilarar
- Discord / póstlisti boð

### Um Kotru (/um-kotru)
- Saga og uppruni kotru
- Af hverju að spila kotru
- Grunnreglur leiksins
- Turneringsreglur (klukka, doubling cube)
- Hugtakaorðalisti IS↔EN

### Kotrusambandið (/felagid)
- Saga félagsins, tilgangur, markmið
- Núverandi stjórnarmenn (nafn, hlutverk, mynd)
- Lög félagsins / starfsreglur
- WBIF aðild, alþjóðleg tengsl

### Spilarar (/spilarar)
- Spilaraskrá með leit og síun
- Röðun / ranking tafla
- Prófíl-sérsíður (sjá 05-spilaraprofilar.md)

### Mót (/mot)
- Dagatal (yfirlit yfir næstu og fyrri mót)
- Tengir beint á WBIF fyrir skráningu og dagskrá
- Úrslit birtast af WBIF/kotra.is

### Fjölmiðlar (/fjolmidlar)
- Safn íslenskra fjölmiðlagreina
- Flokkað eftir: ári, fjölmiðli, tegund

### Póstlisti (/postlisti)
- Listmonk skráningarform
- Arkív eldri fréttabréfa
